Proyecto Java Fundamentos del Lenguaje


Estructura del Proyecto

Main.java
Clase principal del programa. Permite ejecutar los métodos de prueba de las demás clases.

TiposDeDatosPrimitivos.java
Muestra el uso de los tipos de datos primitivos como int, double, char y boolean.

VariablesYConstantes.java
Explica cómo declarar variables y cómo usar constantes con la palabra reservada final.

Operadores.java
Demuestra el uso de operadores aritméticos, relacionales y lógicos.

ConversionDeTipos.java
Ejemplos de conversión de datos entre diferentes tipos usando casting.

EntradaDeDatos.java
Lectura de información desde teclado utilizando la clase Scanner.

EstructurasCondicionales.java
Uso de estructuras de decisión como if, else y switch.

EstructurasDeRepeticion.java
Ejemplos de ciclos for, while y do while.

ArreglosUnidimensionales.java
Manejo de arreglos de una sola dimensión, recorrido y acceso a posiciones.

ArreglosBidimensionales.java
Uso de matrices y recorrido mediante ciclos anidados.

ManipulacionCadenas.java
Operaciones básicas con textos usando la clase String.

MetodosEstaticosSimples.java
Creación y uso de métodos estáticos.

ManejoBasicoExcepciones.java
Ejemplo de manejo de errores con try catch para evitar fallos en la ejecución.

Requisitos
Tener instalado Java JDK 8 o superior y java pack


Ejecucion
1. Abrir el proyecto
2. Ejecutar la clase Main.java.
3. Desde allí se pueden llamar los métodos de cada clase según se necesite probar.

Objetivo
El objetivo de este proyecto es practicar la sintaxis básica de Java
